# Техническое задание

## 1. Описание задачи
Разработать утилиту, которая генерирует **красивый отчёт о системной информации** в терминале с использованием библиотеки `rich`. Отчёт должен включать:
- Таблицу с информацией о платформе (ОС, версия Python, процессор)
- Прогресс-бар имитации «загрузки данных»
- Панель с кратким выводом
- Подсвеченный фрагмент кода примера использования

## 2. Цель работы
Освоить базовые компоненты `rich`: `Console`, `Table`, `Progress`, `Panel`, `Syntax`.

## 3. Функциональные требования
- Использовать `rich.table.Table` для отображения системной информации
- Имитировать загрузку с помощью `rich.progress.Progress`
- Вывести панель с итоговым сообщением
- Показать пример кода с подсветкой синтаксиса
- Весь вывод должен быть цветным и структурированным

## 4. Нефункциональные требования
- Код должен быть читаемым и модульным
- Не использовать `print()` — только `console.print()`
- Поддержка автоматического определения ширины терминала

## 5. Ограничения и допущения
- Разрешено использовать только стандартную библиотеку + `rich`
- Не требуется реальная загрузка данных — только имитация через `time.sleep()`
- Утилита запускается в интерактивном терминале

## 6. Источник ТЗ
Самостоятельная постановка.
